1.1 Мета роботи: Розгорнути програмну систему та продемонструвати її роботу. 

Хід роботи
Встановлення необхідного програмного забезпечення:
1. PostgreSQL (база даних)
2. Node.js (середовище виконання JavaScript)
3. npm (менеджер пакетів Node.js)


Створення бази даних:
1. Створити нову базу даних в PostgreSQL
2. Виконати зазначені в коді SQL-скрипт для створення таблиць і внесення початкових даних
Встановлення залежностей:
Виконати команду npm install для установки необхідних залежностей


Запуск програми:
1. Написати cd та обрати папку з проєктом
2. Написати `npm run dev`
Демонстрація роботи програми:
1. Відкрити веб-браузер і перейти за адресою http://localhost:3000


В якості API використовувалися такі сервіси:
1. EONET_API - це сервіс, який відстежує природні події по всьому світу, такі як землетруси, урагани, лісові пожежі тощо. Він надає дані про місце розташування, час та інші деталі цих подій.
Переваги даного API:
Використовуючи API EONET, можна отримувати оновлення про природні катастрофи в режимі реального часу. Це саме той API, який і потрібен для мого проєкту а саме із надзвичайними ситуаціями, новинами, екологічним моніторингом тощо.
2. USGS_API - це державна організація США, яка займається вивченням природних ресурсів країни. Її API надає дані про землетруси, геологічні дослідження, водні ресурси тощо.
Переваги даного API:
Використовуючи API USGS, можна отримувати дані про землетруси, геологічні дані, інформацію про водні ресурси та багато іншого.
